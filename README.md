# Web Print - Grafikrea

## Resumen del proyecto

Web Print es un sitio web corporativo para **Grafikrea**, enfocado en presentar de forma visual sus servicios de diseno grafico e impresion profesional. El proyecto comunica propuesta de valor, experiencia de marca y canales de contacto a traves de una interfaz moderna, con navegacion por secciones como Inicio, Servicios, Nosotros y Contactos.

## Tecnologias utilizadas

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
- Motion (`motion/react`)
- React Icons
